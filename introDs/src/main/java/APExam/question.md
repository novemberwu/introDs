Q: What is the result of the expression 7 / 2 + 7.0 / 2?
0) 6.5
1) 7.0
2) 6
3) 7.5
   Correct: 0
   Explanation: 7/2 is integer division, which equals 3. 7.0/2 is double division, which equals 3.5. 3 + 3.5 = 6.5.

Q: String s = "Java"; s.concat(" Script"); System.out.println(s); What is printed?
0) Java Script
1) Script
2) Java
3) Compilation Error
   Correct: 2
   Explanation: Strings in Java are immutable. The concat() method returns a new String, but the original variable 's' remains "Java" unless reassigned.

Q: According to De Morgan's Law, which of the following is equivalent to !(A && B)?
0) !A && !B
1) !A || !B
2) A || B
3) !A || B
   Correct: 1
   Explanation: De Morgan's Law states that the negation of an AND is the OR of the negations: !(A && B) becomes !A || !B.

Q: What is the index of the last element in an array 'data' that has a length of 10?
0) 10
1) 0
2) 1
3) 9
   Correct: 3
   Explanation: Array indexing in Java starts at 0, so the last index is always length - 1.

Q: Which of the following correctly describes the relationship "A Car has an Engine"?
0) Inheritance (Car extends Engine)
1) Composition (Car has an Engine instance variable)
2) Interface (Car implements Engine)
3) Polymorphism
   Correct: 1
   Explanation: A "has-a" relationship is represented by composition (fields), while an "is-a" relationship is represented by inheritance.
